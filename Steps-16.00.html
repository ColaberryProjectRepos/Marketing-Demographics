Time Stamp View Instructions 16

1. Create a new Worksheet

2. Name it TimeStamp

3. Create a Calculated Field called _cc.Blank Inside the body just put ‘’   then click the OK button.

4. Drag _cc.Blank into Text(Label)



5. Click on WorkSheet and select Show Captions.


6. Click the Down Arrow in the Caption Box and select Edit Caption.



7. Erase whatever is in there and type Last Refreshed: 
8. Then hit enter and Insert Date Updated Time.


9. Right Click the Title and Select Hide.
